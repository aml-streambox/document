## ⚠️ Important Notes

- System stability is limited - green screen, black screen, or system crashes may occur
- HDMI hot-plug events may cause issues - reboot when problems arise
- This project is created entirely through "vibe coding" using various AI models
- The author has limited understanding of how the AI-generated code works

## 🚀 Key Features

- **4K60fps HDR Passthrough** - Ultra low latency with VRR support
- **Hardware Encoding** - H.264/H.265 up to 4K60fps
- **Streaming Protocols** - SRT, RTMP via GStreamer
- **Web UI** - Cockpit-based management interface
