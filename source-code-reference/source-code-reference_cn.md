本项目 Yocto 源码基于 [Khadas VIM4 Yocto 版本](https://docs.khadas.com/products/sbc/vim4/development/linux/build-yocto)。

## 参考资料

- 文档：https://docs.khadas.com/products/sbc/vim4/development/linux/build-yocto
- SDK 基础：https://dl.khadas.com/resources/development/yocto/khadas_vims_yocto_sdk_base/
